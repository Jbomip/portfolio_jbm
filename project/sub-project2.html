퍼블리싱 가이드라인

  • swiper를 이용한 슬라이드 이미지 구현
    • js
    • 일정 시간이 지날때 마다 슬라이드가 하나씩 넘어감
      - autoplay : {delay:2500}(2.5s) 추가
    • 슬라이드가 연속해서 돌아감
      - loop : true 추가
    • 보이는 슬라이드 개수 4개
      - view를 4로 변경
    • css
    • 이전, 다음 버튼 이미지 변경
      - .swiper-button-next, .swiper-botton-prev
      영역 지정 후 background-image 적용
  • 슬라이드를 클릭하면 화면이 위로 올라감
    • js
    • 적정 박스에 offset top부분을 선언
      클릭시 scrollTo 선언 위치
      부드럽게 올라가도록 behavior : ”smooth”추가
  • 슬라이드 클릭시 위에 내용이 클릭한 내용으로 바뀜
    • html 처음칸 제외 나머지 칸에 클래스 none 추가
    • css 클래스 none display:none, 클래스 active display:block 추가
    • js 클릭한 부분에 addClass(“active”) 나머지 removeClass(“active”) 적용

문제 & 해결

  • 슬라이드 클릭시 위에 내용이 클릭한 내용으로 바뀜부분에서

    • 문제점

      display: block을 주니 구조가 무너짐

      .

      어려웠던 점

      클래스 active만 주고 다른 슬라이드를 누르니
      기존 칸이 사라지지않음

      다른 칸을 누르면 처음 칸이 사라지지 않음

    • 해결

      클래스 none의 하위자식을 감싸주는 부모를 만듦

      .

      해결

      클릭시에 나머지 칸에 모두 removeClass(“active”)를 추가해줌

      다른칸 클릭시 처음칸에 removeClass(“active”) 후 addClass(“none”)을 추가해줌
